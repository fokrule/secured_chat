Computer Security Secured Chat Project

Steps

Here are the steps and details for this secured chat project:

  • Implemented Handshake.
  • Created RSA keys for two different users. In future development, they will share their public keys separately instead of creating them here and will use their respective private keys to decrypt messages.
  • Each message is encrypted before sending and decrypted after receiving.
  • Client 1 uses Client 2's public key to encrypt, and Client 2 uses Client 1's public key.
  • Client 1 uses their own private key to decrypt messages, and Client 2 uses their own private key to decrypt.
